# Ebytr

Projeto exemplo usado para mentoria de carreira.

## Stack utilizada

**Back-end:** Node, Express, MySQL2

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/lucas-santos/ebytr
```

Entre no diretório do projeto

```bash
  cd ebytr
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
